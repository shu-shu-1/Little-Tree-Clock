# 支持与帮助 (Support)

如果您在使用 **小树时钟 (Little-Tree-Clock)** 的过程中遇到任何问题，或者想要获取相关帮助，请参考以下渠道。

## 📖 官方文档与资源
在提问之前，请查阅已有的项目指南。这也许能解决您 80% 的问题：
* **[README 首页](README.md)**：包含安装说明和快速启动步骤。
* **[插件指南](PLUGIN_GUIDE.md)**：包含关于如何安装、使用以及开发自定义插件的详细信息。

## 🐛 报告 Bug
如果您确认自己发现了一个程序错误、崩溃，或某些功能未按预期工作，请使用 [GitHub Issues](../../issues) 板块来报告它。

提交 Bug 时请提供：
- 您的系统环境（Windows/macOS/Linux 等及具体版本）
- 错误日志或终端输出（如：`uv run main.py` 时的报错）
- 提供具体的重现步骤可以大大加快修复进度。

## 💡 提出新功能建议
有一个好点子可以让小树时钟变得更好？非常欢迎！
您同样可以在 [GitHub Issues](../../issues) 中创建一个 "Feature Request"。请描述您的想法以及适用的场景。如果在您的需求属于更定制化的功能，不妨考虑开发一款 [第三方插件](PLUGIN_GUIDE.md)。

## ❓ 提问与交流
目前我们通过 GitHub 平台进行交流。您可以随时通过开 Issue（使用 `question` 标签）来咨询配置、运行或任何项目相关的问题。如果你需要更直接的帮助，可以通过电子邮件 `zsxiaoshu@outlook.com` 联系我。

感谢您对小树时钟的喜爱和支持！
