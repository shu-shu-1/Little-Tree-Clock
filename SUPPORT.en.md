# Support

If you encounter any issues or need help while using **Little-Tree-Clock**, please refer to the following channels.

## 📖 Official Documentation & Resources
Before asking questions, please consult the existing project guides. These might solve 80% of your problems:
* **[README Home](README.en.md)**: Contains installation instructions and quick start steps.
* **[Plugin Guide](PLUGIN_GUIDE.md)**: Details on how to install, use, and develop custom plugins.

## 🐛 Bug Reports
If you confirm finding a program bug, crash, or features that don't act as expected, please use the [GitHub Issues](../../issues) section to report it.

When submitting a bug, please provide:
- Your system environment (Windows/macOS/Linux & specific version)
- Error logs or terminal output (e.g., error when running `uv run main.py`)
- Providing specific reproduction steps vastly speeds up the fix.

## 💡 Feature Requests
Got a great idea to make Little Tree Clock better? Welcome aboard!
You can also create a "Feature Request" in [GitHub Issues](../../issues). Please describe your idea and the application scenario. If your needs are highly customized, perhaps consider developing a [third-party plugin](PLUGIN_GUIDE.md).

## ❓ Questions & Discussions
We currently communicate via the GitHub platform. You can comfortably use Issues (with the `question` tag) to consult regarding configuration, operation, or any project-related query. If you need direct assistance, you can also reach out via email at `zsxiaoshu@outlook.com`.

Thank you for your love and support for Little Tree Clock!
